

## /api/v1/auth/signup

    register a new  user

    method: post

    json_req example:
    {
        "email": "abc@abc.com",
        "password": "1234",
        "role": "Student"                   
    }

     //role options ['Teacher', 'Student']



    json response example:
    {
    "success": true
    }


## /api/v1/auth/login

    returns jwt token  as login.

    method: post

    json_req example:
    {
        "email": "abc@abc.com",
        "password": "1234",
                   
    }

    json response example:
    {
    "success": true,
    "token": "jwt token"
}



    ## /api/v1/teacher/studentPhoto

    adds studentImage  

    method: post

    
    multipart  formdata request
    {
        filename: studentImage,
        fileType: jpg/png/
    }

    json response example:
     {
    "success": true
    }
